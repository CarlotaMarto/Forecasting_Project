1. Introduction

This project analyzes and forecasts monthly marriages in Queensland, Australia, using the Box-Jenkins methodology (ARIMA/SARIMA).
The dataset is sourced from the Queensland Government Open Data portal (Marriages by Month), which provides official monthly counts of marriages registered across the state.

The objective of this analysis is to identify trends and seasonal patterns in the number of marriages, fit a suitable SARIMA model, and produce reliable forecasts for the next 12 months.
These forecasts can provide insights for government planning, service provision, and understanding social trends related to marriage in Queensland.

As the variance is not constant over time, we need to stabilize it. For that, we will use the Box-Cox transformation with the guerrero method for finding lambda.

lambda <- data_train %>%
  features(Marriages, features = guerrero) %>%
  pull(lambda_guerrero)
lambda
## [1] 0.4751842
data_train <- data_train %>%
  mutate(TransformedMarriages = box_cox(Marriages, lambda))
